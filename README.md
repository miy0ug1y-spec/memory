# memory

## サイト概要
### サイトテーマ
大切な人を失った人の想いを投稿するコミュニティサイト
​
### テーマを選んだ理由
大切な人との別れを経験した際、気持ちや想いを整理するための手段の重要性を感じました。<br>
現在は、行政機関や民間サービスによる電話・メール相談、面談予約などの支援があります。<br>
一方で、すぐに誰かへ相談するのではなく、まずは自身の気持ちを整理したいと考える方も一定数存在すると考えています。

本テーマでは、遺族の心理的負担の軽減やグリーフケアの一助となるよう、気持ちや想いを<br>
安心して記録・整理できるサービスを想定しました。<br>
また、前々職では生命保険業務を通じて、ご逝去後の各種手続きやご契約のアフターフォローに携わっていました。<br>
その経験から、情報提供だけではなく、感情整理を支援する場の必要性を感じ、本テーマを選定しました。

### ターゲットユーザ
・大切な人を失った気持ちを一人で整理したい人​<br>
・心理的負担を誰かに相談したい人<br>
・整理した気持ちを遺す人に伝えたい人<br>

### 主な利用シーン
・大切な人への想いを投稿する時 <br>
・投稿にコメントや共感(いいね)する時 <br>
・自分と共感する人を検索する時 <br>
