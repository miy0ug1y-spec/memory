# memory

## サイト概要
### サイトテーマ
亡くなった大切な人への想いを投稿するコミュニティサイト
​
### テーマを選んだ理由
私の母は突然亡くなりました。 <br>
気持ちのやり場が分からないなか自分の気持ちをまとめると心が少し楽になりました。

行政などのサービスでメールや電話相談、個人面談の予約ができるものはあります。しかし <br>
自分の気持ちを整理したい、ただ想いをとどめておきたいと思ったときに <br>
気兼ねせず使えるサービスがあったら良いと思いました。

私は前職で生命保険を通じて最期のお手続きをさせていただいていました。 <br>
その時も遺された側に対して感じたことは <br>
誰かに相談したいわけではなく、何かしてほしいわけでもないのだということです。

お客様の心に寄り添う情報提供として、このようなサイトをご案内できればよかったという <br>
気持ちも込めてテーマに選びました。

### ターゲットユーザ
・大切な人を亡くしたばかりの人 <br>
・大切な人を亡くしてから気持ちの整理がつかない人
​
### 主な利用シーン
・大切な人への想いを投稿する時 <br>
・投稿にコメントや共感(いいね)する時 <br>
・自分と共感できそうな人を検索する時 <br>
